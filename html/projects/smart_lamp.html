Smart Lamp

Smart Lamp This is my very first project: a smart lamp controlled by voice using Arduino. It serves as my introduction to schematics in a practical way, combining electronics, coding, and 3D design to bring an idea to life.

Project Inspiration

The goal was to create a functional and interactive lighting system that responds to voice commands, merging hardware and software for a seamless user experience.

Project Features

  • Voice Control: Uses speech recognition to turn the lamp on and off.
  • Arduino Integration: The microcontroller processes commands and controls the LED system.
  • 3D Design: A custom-designed lamp structure, modeled and printed for a sleek, modern look.
